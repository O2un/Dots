# Dots 프로젝트

**Unity DOTS(Data-Oriented Technology Stack) 기반 고성능 프레임워크**

## 📦 Libraries (라이브러리 구성)

### Open Source Libraries
| Library | Version | Description | License |
| :--- | :--- | :--- | :--- |
| **[UniTask](https://github.com/Cysharp/UniTask)** | Latest | Unity를 위한 고성능 비동기 처리 라이브러리 | MIT |
| **[R3](https://github.com/Cysharp/R3)** | Latest | 차세대 반응형 프로그래밍(Reactive Extensions) 프레임워크 | MIT |

### Paid Libraries (유료 라이브러리)
> **주의**: 아래 라이브러리는 라이선스 정책에 따라 저장소에 포함되지 않을 수 있으며, Unity Asset Store를 통해 별도 구매가 필요합니다.

* **[Odin Inspector & Serializer](https://odininspector.com/)**: 강력한 에디터 확장 및 데이터 직렬화 도구.

## 🚀 Getting Started (시작하기)

### Requirements
* Unity 2022.3 LTS 이상
* Entities 1.0.0 이상
* Burst 1.8.0 이상

### Installation
1.  이 저장소를 Clone 합니다.
    ```bash
    git clone https://github.com/O2un/Dots.git
    ```
2.  Unity 프로젝트를 열고, 패키지 매니저를 통해 필요한 의존성(Entities, Burst 등)이 설치되었는지 확인합니다.
